## Running the app in development
1. Clone the repository
2. Run dependency installation 
```bash
npm install
```
3. Have Nest CLI installed
```bash
npm i -g @nestjs/cli
```
4. Run database
```bash
docker-compose up -d
```

## Running the database
```bash
docker-compose up -d
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
